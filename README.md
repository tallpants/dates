# dates

Try it out: https://tallpants.github.io/dates

## Why

Date difference tools online are clunky, annoying, and slow (e.g: [timeanddate.com](https://www.timeanddate.com/date/duration.html), [calculator.net](https://www.calculator.net/date-calculator.html)). There's a handful of ops I do frequently, so I made this for myself.

## Features

- Diff 2 dates in different units of time
- Convert to / from UNIX epoch (epoch fields are editable inputs, date fields are synced to epoch fields)
- Add / subtract different units of time from a date
- Autofocus input on page load
- Pre-fills the current date where it makes sense.

## Non-Features

- No styling
- No ads or analytics
- No API for business date calculators
- No accounts
- No "Love our site? Please support us" callout
- No "History of the Gregorian Calendar" write up
- No US federal holidays list
