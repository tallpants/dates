# dates

Try it out: https://tallpants.github.io/dates

## Why

Date difference tools online are clunky, annoying, and slow (e.g: https://www.timeanddate.com/date/duration.html, and https://www.calculator.net/date-calculator.html). There's a handful of ops I do frequently, so I made this for myself.

## Features

- Get difference between 2 dates in different units of time
- Convert timestamps to / from UNIX epoch
- Add / subtract different units of time from a date

## Non-Features

- No styling
- No ads or analytics
- No API for business date calculators
- No accounts
- No "Love our site? Please support us" callout
- No "History of the Gregorian Calendar" write up
- No US federal holidays list
